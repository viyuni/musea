# Card

`Card` is a simple content container for grouped information.

## Recommended usage

- Wrap related text or actions in one visual block
- Use the title slot for clear section labels
- Keep body text concise for scanability

## Slots

| Slot      | Description       |
| --------- | ----------------- |
| `title`   | Header/title area |
| `default` | Main content body |

## Showcase variants

- `Basic`: minimal title + short body
- `With Body`: title with longer supporting copy
